Name

default.pa - PulseAudio Sound Server Startup Script

Synopsis

~/.config/pulse/default.pa

/pulseaudio/etc/pulse/default.pa

/pulseaudio/etc/pulse/system.pa

Description

The PulseAudio sound server interprets a configuration script on startup, which is mainly used to define the set of modules to load. When PulseAudio runs in the per-user mode and ~/.config/pulse/default.pa exists, that file is used. When PulseAudio runs in the per-user mode and that file doesn't exist, /pulseaudio/etc/pulse/default.pa is used. When PulseAudio runs as a system service, /pulseaudio/etc/pulse/system.pa is used.

The script should contain directives in the PulseAudio CLI language, as documented in pulse-cli-syntax(5).
